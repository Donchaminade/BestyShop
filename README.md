# ZakSport

## Présentation du Projet

Une plateforme e-commerce moderne et réactive dédiée à la vente de produits sportifs. ZakSport offre une expérience utilisateur fluide pour la navigation, la sélection et l'achat de produits, ainsi qu'une interface d'administration robuste pour la gestion du catalogue.

## Fonctionnalités Clés

*   **Catalogue Produits :** Explorez une large gamme d'articles sportifs avec des détails complets.
*   **Panier d'Achat :** Gérez facilement vos sélections et procédez à l'achat.
*   **Authentification Utilisateur :** Inscription et connexion sécurisées.
*   **Page d'Administration :**
    *   Gestion complète des produits (ajout, modification, suppression).
    *   Visualisation paginée des produits (25 par page).
*   **Pages Informatives :** Sections "À propos" et "Contact" pour les informations sur l'entreprise et le support.

## Technologies Utilisées

Ce projet est construit avec les technologies modernes suivantes :

*   **Frontend :** React.js (avec TypeScript)
*   **Styling :** Tailwind CSS
*   **Build Tool :** Vite
*   **Backend & Base de Données :** Supabase (pour l'authentification, la base de données et les APIs)
*   **Gestion d'État / Requêtes :** React Query (pour la gestion des données asynchrones)
*   **Icônes :** Lucide React

## Comment Démarrer le Projet

Pour démarrer le projet localement, suivez ces étapes :

1.  **Cloner le dépôt :**
    `git clone [URL_DU_DEPOT]`
    `cd Zak_Sport`

2.  **Installer les dépendances :**
    `npm install`

3.  **Configurer Supabase :**
    Assurez-vous d'avoir un projet Supabase configuré. Créez un fichier `.env` à la racine du projet avec vos clés Supabase :
    ```
    VITE_SUPABASE_URL=YOUR_SUPABASE_URL
    VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
    ```
    *(Note : Remplacez `YOUR_SUPABASE_URL` et `YOUR_SUPABASE_ANON_KEY` par les véritables valeurs de votre projet Supabase.)*

4.  **Lancer l'application :**
    `npm run dev`
    L'application sera accessible sur `http://localhost:5173` (ou un port similaire).
